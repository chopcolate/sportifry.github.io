components:
- thư mục này chứa các components để sử dụng trong app
- header, navbar, footer,...
- tìm hiểu react component: class component và function component để biết cách viết

css:
- chứa các css của từng component
- giả sử có một component tên header.js thì trong css sẽ có một file tên header.css
- viết như css bình thường

util:
- utility: chứa code dùng để hỗ trợ (connect tới firebase)
- sử lý các event: login, logout,...
- tìm hiểu firebase firestore, realtime, react + firebase,...
